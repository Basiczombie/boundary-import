Boundary Import

FAQ

  • About the Tool:
    Boundary Import is a tool that takes a .KML (Keyhole Markup Language) file that can be exported from google maps and parses it into a Polyline readable by PowerSchool.

  • What is a .KML?
    A .KML file is a file that uses XML to express geographic annotation and visualization by storing locations, image overlays, video links and modeling information like lines, shapes, 3D images and points.

  • What is a Polyline?
    Polyline encoding is a lossy compression algorithm that allows you to store a series of coordinates as a single string. The encoding process converts a binary value into a series of character codes for ASCII characters.

Instructions

  1. Create and export a .KML file from Google Maps.

  2. Select file to import.

  3. Verify the Polyline string has been generated in Boundary textfield. If a string is already present check to see if that sting has been replaced.

  4. Click Submit.

  5. Click Edit Boundary and verify school boundary is correct.
